# TaskDetailPageContent Component Refactoring

## Структура компонента

Компонент `TaskDetailPageContent` был разбит на несколько меньших компонентов для улучшения читаемости и поддерживаемости кода.

## Созданные компоненты

### 1. `types.ts`
Содержит все типы TypeScript, используемые в компоненте:
- `Task` - основной тип задачи
- `TaskStatus` - статусы задачи

### 2. `utils.ts`
Содержит утилитарные функции:
- `statusColors` - цвета для статусов
- `getStatusName` - получение названия статуса
- `getUserProfileLink` - получение ссылки на профиль пользователя

### 3. `DisputeForm.tsx`
Компонент формы для создания спора по задаче.

### 4. `TaskHeader.tsx`
Компонент заголовка задачи с описанием, автором и кнопками действий.

### 5. `TaskInfoPanel.tsx`
Компонент информационной панели со статусом, ценой и категорией.

### 6. `TaskFiles.tsx`
Компонент для отображения файлов задачи.

### 7. `DisputeStatus.tsx`
Компонент для отображения статуса спора.
